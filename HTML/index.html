Components

Shippd CSS also includes some slideshow components. To use these include the JS files at the bottom of the body and use the markup as shown.

						
<div>
	<shippd-slides time="5000" transition="300" width="259" height="183">
		<img src="../images/kitten-0.jpg" width="259" height="183">
		<img src="../images/kitten-1.jpg" width="259" height="183">
		<img src="../images/kitten-2.jpg" width="259" height="183">
		<img src="../images/kitten-3.jpg" width="259" height="183">
		<img src="../images/kitten-4.jpg" width="259" height="183">
	</shippd-slides>
</div>

Include this code at the bottom of your html doc:

							
<body>
	...
	<script src="./js/shippd-slides.js"></script>
</body>
